Описание проекта.
Интернет-магазин «В один клик» продаёт разные товары: для детей, для дома, мелкую бытовую технику, косметику и даже продукты. 
Отчёт магазина за прошлый период показал, что активность покупателей начала снижаться.
Необходимо разработать решение, которое позволит персонализировать предложения постоянным клиентам, 
чтобы увеличить их покупательскую активность.

Для решения поставленной задачии, были протестированны 4 модели (DecisionTreeClassifier, KNeighborsClassifier, LogisticRegression, SVC). 
Лучший результат показала модель логистической регрессии, ее метрика roc auc равна примерно 0.9.
